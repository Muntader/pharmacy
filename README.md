#Pharmacy Management System
A comprehensive pharmacy management system built with Laravel 5.5, originally released on CodeCanyon in 2017.

#About
This was my first commercial project on CodeCanyon, launched in 2017. During its availability, it was successfully sold to 42 clients before being removed from the marketplace in 2018.

#Features
Medicine Management - Track and manage medicinal inventory with categories
Point of Sale (POS) - Complete POS system for pharmacy sales
Supplier Management - Manage supplier information and orders
Customer Management - Track customer details and purchase history
Sales Tracking - Comprehensive sales reports and analytics
Multi-User Support - User roles with different permission levels
Order Management - Handle customer orders efficiently
API Integration - RESTful API with Passport authentication
Excel Export - Export data to Excel spreadsheets
Tech Stack
Laravel 5.5
PHP 7.0+
Laravel Passport (API Authentication)
Laravel Cashier (Payment Processing)
Maatwebsite Excel (Data Export)
Legacy Project
This is a legacy project from 2017 and is no longer actively maintained or sold commercially.
