# Pharmacy Management System

A comprehensive pharmacy management system built with Laravel 5.5, originally released on CodeCanyon in 2017.

## About

This was my first commercial project on CodeCanyon, launched in 2017. It sold to 42 clients before being removed from the marketplace in 2018.

## Features

- **Medicine Management** – Track and manage inventory with categories  
- **Point of Sale (POS)** – Complete POS system for pharmacy sales  
- **Supplier Management** – Manage supplier info and orders  
- **Customer Management** – Track customer details and purchase history  
- **Sales Tracking** – Comprehensive sales reports and analytics  
- **Multi-User Support** – Role-based permissions  
- **Order Management** – Efficient order handling  
- **API Integration** – RESTful API with Passport authentication  
- **Excel Export** – Export data via Maatwebsite Excel  

## Tech Stack

- Laravel 5.5  
- PHP 7.0+  
- Laravel Passport (API auth)  
- Laravel Cashier (payments)  
- Maatwebsite Excel (export)

## Status

Legacy project (2017) – no longer maintained or sold commercially.
